# vkBot
###### Бот для сайта vk.com, написанный на javascript с использованием nodejs
###### Проверить бота можно на страничке https://vk.com/id294418496

## Как использовать
1. Скачать бота
2. Изменить логин и пароль в файле bot-vk.js
3. Запустить (node launch.js)
